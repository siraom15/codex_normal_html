แสดง/ซ่อน เมนู


การกำหนดรูปแบบเว็บไซต์ใน HTML ( HTML Layout )

ตัวอย่างการจัดหน้าเว็บไซต์

องค์ประกอบของเค้าโครง HTML

เว็บไซต์มักจะแสดงเนื้อหาในหลายคอลัมน์ (เหมือน นิตยสารหรือหนังสือพิมพ์)

ใน HTML5 มี element ที่มีความหมาย กำหนดส่วนต่าง ๆ ของหน้าเว็บ เช่น

  • <header> - กำหนดส่วนหัวสำหรับเอกสาร หรือส่วนของเอกสาร
  • <nav> - กำหนด ลิ้งค์ การนำทาง
  • <section> - ใช้ mark up เนื้อหาที่ต้องการแยกออกมาเป็นส่วนๆ เช่น การแบ่งบทความออกเป็น chapter
  • <article> - ใช้ mark up สิ่งที่มีความหมายในตัวของมันเอง เช่น ข่าว บทความ เป็นต้น
  • <aside>- จะใช้ mark up ส่วนที่มีความเกี่ยวข้องกับเนื้อหาหลัก แต่ไม่มากนัก จนสามารถแยกออกมาจากเนื้อหาหลักหรือตัดออกได้
  • <footer> - กำหนดส่วนท้ายสำหรับเอกสารหรือส่วนของเอกสาร
  • <details> -กำหนดรายละเอียดเพิ่มเติม
  • <details> กำหนดหัวเรื่องสำหรับองค์ประกอบ <details>


เทคนิคการจัดวางหน้าเว็บ

มี 5 วิธีในการสร้างเค้าโครงหลายคอลัมน์ แต่ละวิธีมีข้อดีข้อเสีย :

  • HTML tables (ไม่แนะนำ)
  • CSS float property
  • CSS flexbox
  • CSS framework
  • CSS grid

เลือกแบบไหนดีนะ ?

- ตาราง (HTML tables)

องค์ประกอบ <table> ไม่ได้ถูกออกแบบมาให้เป็นเครื่องมือในการจัดวาง

วัตถุประสงค์ขององค์ประกอบ <table> คือการแสดงข้อมูลแบบตาราง

ดังนั้นอย่าใช้ตารางสำหรับเค้าโครงหน้าของคุณ!

เคล็ดลับ: อย่าใช้ตารางสำหรับเค้าโครงหน้าของคุณ!
- CSS Framework

หากคุณต้องการสร้างเลย์เอาต์ของคุณอย่างรวดเร็วคุณสามารถใช้เฟรมเวิร์กเช่น Bootstrap หรือ Semantic ui

CSS Floats

เป็นเรื่องปกติที่จะทำทั้งโครงร่างเว็บโดยใช้คุณสมบัติของ CSS float เพราะง่ายต่อการเรียนรู้

คุณต้องจำไว้ว่าคุณสมบัติของ Floats นั้นทำงานอย่างไร

ข้อเสีย: องค์ประกอบ Float จะเชื่อมโยงกับผังเอกสาร ซึ่งอาจมีผลต่อความยืดหยุ่นของเว็บไซต์
สามารถศึกษาเรื่อง CSS Floats ได้ ที่นี่

CSS Flexbox

Flexbox เป็น Layer ใหม่ใน CSS3

การใช้ flexbox ช่วยให้มั่นใจได้ว่าองค์ประกอบต่างๆจะทำงานได้อย่างที่คาดการณ์ไว้

เมื่อเค้าโครงหน้าต้องรองรับขนาดหน้าจอที่แตกต่างกันและอุปกรณ์แสดงผลที่แตกต่างกัน

ข้อเสีย: ไม่ทำงานใน IE10 และรุ่นก่อนหน้า

CSS Grid View

CSS Grid Layout Module นำเสนอระบบโครงร่างแบบกริดที่มีแถวและคอลัมน์

ทำให้การออกแบบหน้าเว็บง่ายขึ้นโดยไม่ต้องใช้ Floats และการวางตำแหน่ง

ข้อเสีย: ไม่ทำงานใน IE หรือ Edge 15 และรุ่นก่อนหน้า

Layer คืออะไร ?
layer คือ ชั้น ของการซ้อนทับกันของภาพเป็นชั้นๆ
layer นั้นเปรียบเสมือนเเผ่นใส่ ที่เมื่อเรานำมาวางทับกัน
ก็จะสามารถมองเห็น
ภาพที่เกิดจากการซ้อนกันของแผ่นใสได้เหมือนเป็นภาพเดียว
Layer กืคือสิ่งที่ใช้บรรจุ ออปเจ็ค เช่นภาพ , ตัวหนังสือ ฯลฯ
Layer สามารถบรรจุออปเจ็กต่างๆ ได้ เช่น ภาพ ,ตัวอักษร ฯลฯ
คุณสามารถวางมันลงบนเอกสาร HTML และคุณสามารถเคลื่อนมันไปรอบๆ ถ้าคุณทำการเคลื่อน Layer
นั่นหมายความว่าออปเจ็กที่อยู่ใน Layer ได้เคลื่อนตามไปด้วย
