หัวข้อใน HTML (HTML Headings)

ตัวอย่างรายการของ Headings

หัวข้อ 1

หัวข้อ 2

หัวข้อ 3

หัวข้อ 4

หัวข้อ 5

หัวข้อของ HTML จะถูกกำหนดด้วยแท็ก <h1> ถึง <h6>

โดยที่:
<h1> กำหนดหัวเรื่องที่สำคัญที่สุด

<h6> กำหนดหัวเรื่องที่สำคัญน้อยที่สุด

ตัวอย่างตาม Code ด่านล่าง

<h1>หัวข้อ 1</h1>
<h2>หัวข้อ 2</h2>
<h3>หัวข้อ 3</h3>
ลองเขียนด้วยตนเอง !


การแบ่งเรื่องในหน้า

แท็ก <hr> กำหนดตัวแบ่งเรื่องในหน้า HTML และมักปรากฏเป็นแนวนอน

องค์ประกอบ <hr> ใช้เพื่อแยกเนื้อหาในหน้า HTML เช่นตัวอย่างด้านล่าง :

หัวข้อ 1

ข้อความ

หัวข้อ 2

ข้อความ


Code :
<h1>หัวข้อ 1</h1>
<p>ข้อความ </p>
<hr>
<h2>หัวข้อ 2 </h2>
<hr>
ลองเขียนด้วยตนเอง !
