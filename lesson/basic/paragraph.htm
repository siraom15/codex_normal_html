ย่อหน้าของ HTML (HTML Paragraphs)

ย่อหน้า HTML ถูกกำหนดด้วยแท็ก : <p>

ตัวอย่างตาม Code ด่านล่าง

paragraph.

This is another paragraph.


Code :
<!DOCTYPE html>
<html>
<body>

<p> paragraph.</p>
<p>This is another paragraph.</p>

</body>
</html>
ลองเขียนด้วยตนเอง !

ตัวแบ่งบรรทัด HTML

ใช้ <br> หากคุณต้องการให้ตัวแบ่งบรรทัด (บรรทัดใหม่) โดยไม่ต้องเริ่มย่อหน้าใหม่ ตัวอย่างเช่น:

นี้คือ
ย่อหน้า
ที่ใช้ตัวแบ่งบรรทัด


Code :
<p>นี้คือ<br>ย่อหน้า<br>ที่ใช้ตัวแบ่งบรรทัด</p>
ลองเขียนด้วยตนเอง !
