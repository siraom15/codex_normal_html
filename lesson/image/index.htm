แทรกรูปภาพใน HTML (HTML Image)

การแทรกรูปภาพ

การแทรกรูปภาพ HTML ถูกกำหนดด้วยแท็ก : <img>

ไฟล์ต้นฉบับของรูปภาพ( src="")

ข้อความอธิบาย ( alt)

การระบุคุณลักษณะสามารถใช้ ( width="" ) ระบุความกว้าง และ ( height="" ) ระบุความสูง

ตัวอย่างตาม Code ด่านล่าง

<!DOCTYPE html>
<html>
<body>

<img src="assets/pic/codex.png" alt="the code X" width="100" height="100">

</body>
</html>
ลองเขียนด้วยตนเอง !


การใส่ URL ให้รูปภาพ

แอตทริบิวต์ src ระบุ URL (ที่อยู่เว็บ) ของภาพ:

<img src="url">


ขนาดภาพ - ความกว้างและความสูง


คุณสามารถใช้แอตทริบิวต์ style เพื่อระบุความกว้างและความสูงของรูปภาพ หรือคุณสามารถใช้คุณสมบัติความกว้าง widthและความสูง height ได้:

<img src="img_girl.jpg" alt="Girl in a jacket" width="500" height="600">
ลองเขียนด้วยตนเอง !


รูปภาพบนเซิร์ฟเวอร์อื่น


คุณสามารถแทรกรูปภาพจากเว็ปไซต์อื่นได้โดยการใส่ URL ของรูปภาพไปใน src ตัวอย่างเช่น

<img src="http://www.sukhon.ac.th/site/wp-content/uploads/2017/08/headsite4.png" alt="sukhon.ac.th">
ลองเขียนด้วยตนเอง !


การลิ้งค์รูปภาพ


ลองกดที่รูปภาพ

หากต้องการใช้รูปภาพเป็นลิงก์ให้วางแท็ก <img> ไว้ในแท็ก <a> ตามตัวอย่างด่านล่าง:

<a href="https://www.google.com">
  <img src="https://www.google.com/images/branding/googlelogo/1x/googlelogo_color_272x92dp.png" alt="Google Logo" >
</a>
