การเชื่อมโยงใน HTML ( HTML Links)

คุณจะพบลิ้งค์ในเกือบทุกหน้าของเว็บไซต์ ให้ผู้ใช้ Click เพื่อเปลี่ยนหน้าหนึ่งไปอีกหน้าหนึ่ง

รูปแบบของการลิ้งค์

ใน HTML ลิงก์จะถูกกำหนดด้วยแท็ก <a> :

ปลายทางของลิงค์ถูกระบุในแอตทริบิวต์ href

เช่น ต้องการให้ลิ้งค์ไปสู่ website google.com สามารถเขียนได้เป็น <a href="https://www.google.com/">

ตัวอย่างตาม Code ด่านล่าง

<a href="url">link text</a>
หรือ:
<a href="http://codex.sukhon.ac.th">ไปยังหน้าแรก </a>
ลองเขียนด้วยตนเอง !


ลิ้งค์เว็บไซต์ภายใน (Local Links)

จะเห็นว่า ตามตัวอย่างด่านบนจะเป็นการลิ้งค์ด้วย URL แบบเต็ม คือ https://.....

แต่ถ้าเป็นการลิ้งค์ภายในเว็บไซต์เดียวกัน เราไม่จำเป็นต้องใส่ https: // www .... ตามตัวอย่างด่านล่าง

<a href="hello.html">ลิ้งค์เว็บไซต์ภายใน </a>
ลองเขียนด้วยตนเอง !


ลิ้งค์เว็บไซต์ภายนอก (External Links)

แต่ถ้าเป็นการลิ้งค์ภายนอก เราจำเป็นต้องใส่ URL แบบเต็ม ( https: // www ....) หรือด้วยเส้นทางที่สัมพันธ์กับหน้าเว็บปัจจุบัน ตามตัวอย่างด่านล่าง

<a href="http://codex.sukhon.ac.th">CodeX </a>
ลองเขียนด้วยตนเอง !

หรือ :

<a href="https://www.google.com">Google website </a>
ลองเขียนด้วยตนเอง !
