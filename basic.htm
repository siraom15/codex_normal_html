พื้นฐาน HTML

เอกสาร HTML ทั้งหมดจะต้องเริ่มต้นด้วยการประกาศประเภทเอกสาร: <!DOCTYPE html>

เอกสาร HTML นั้นเริ่มต้นด้วย <html> และลงท้ายด้วย </html>

ส่วนที่มองเห็นได้ของเอกสาร HTML อยู่ระหว่าง <body> และ </body>

ตัวอย่างตาม Code ด่านล่าง

<!DOCTYPE html>
<html>
<body>

<h1>My First Heading</h1>
<p>My first paragraph.</p>

</body>
</html>

ส่วนหัวของ HTML (HTML Headings)

ส่วนหัว HTML ถูกกำหนดด้วยแท็ก <h1> ถึง <h6>

โดยที่ <h1> กำหนดหัวเรื่องที่สำคัญที่สุด <h6> กำหนดหัวเรื่องที่สำคัญน้อยที่สุด

ตัวอย่างตาม Code ด่านล่าง


<h1>This is heading 1</h1>
<h2>This is heading 2</h2>
<h3>This is heading 3</h3>

ลองเลย !

ย่อหน้าของ HTML (HTML Paragraphs)

ย่อหน้า HTML ถูกกำหนดด้วยแท็ก : <p>

ตัวอย่างตาม Code ด่านล่าง


<!DOCTYPE html>
<html>
<body>

<p>This is a paragraph.</p>
<p>This is another paragraph.</p>

</body>
</html>
ลองเลย !

ลิ้งค์ HTML (Links HTML)

ลิงค์ HTML ถูกกำหนดด้วยแท็ก : <a>

ปลายทางของลิงค์ถูกระบุในแอตทริบิวต์ href

เช่น ต้องการให้ลิ้งค์ไปสู่ website google.com สามารถเขียนได้เป็น <a href="https://www.google.com/">

ตัวอย่างตาม Code ด่านล่าง


<!DOCTYPE html>
<html>
<body>

<a href="#" >This is a link.</a>

</body>
</html>
ลองเลย !

แทรกรูปภาพใน HTML (HTML Image)

การแทรกรูปภาพ HTML ถูกกำหนดด้วยแท็ก : <img>

ไฟล์ต้นฉบับของรูปภาพ( src=""), ข้อความอธิบาย ( alt ),การระบุคุณลักษณะสามารถใช้ width ระบุความกว้าง และ height ระบุความสูง

ตัวอย่างตาม Code ด่านล่าง


<!DOCTYPE html>
<html>
<body>

<img src="assets/pic/codex.png" alt="the code X" width="100" height="100">

</body>
</html>
ลองเลย !
